Artificial neural structures (ARNs) are a program for implementing the neural structures of our brain. We will not discuss our knowledge, study the brain, put neurons, which are a kind of organic switches.

Artificial intelligence (AI) is a branch of computational linguistics and computer science that deals with the formalization of problems and tasks that are similar to human actions.

Artificial neural networks mimic the behavior of the brain in a simpler form. They can be taught in controlled and uncontrolled ways. In a controlled ANN, the network is trained by transmitting relevant input information and examples of output information. For example, a spam filter in an e-mail box: the input information can be a list of words that are usually contained in spam messages, and the output information can be a classification for the message (spam or not spam).

The biological neuron is simulated in the ANN through the activation function. In classification tasks (such as spam detection), the activation function must have a "switch" characteristic. In other words, if the input is greater than some value, the output must change state, for example from 0 to 1 or -1 to 1. This mimics the "inclusion" of a biological neuron. The sigmoid function is usually used as an activation function.

From the graph you can see that the function is "activation" - it grows from 0 to 1 with each increase in the value of x. Sigmoid function is smooth and continuous. This means that the function has a derivative, which in turn is a very important factor for learning the algorithm.

As mentioned earlier, biological neurons are hierarchically connected in networks where the output of some neurons is the input for other neurons. We can represent such networks in the form of connected layers with nodes. Each node receives a weighted input, activates an activation function for the sum of inputs, and generates an output.

In the figure above, you can see three layers of the network - Layer 1 is the input layer, where the network receives external input. Layer 2 is called the hidden layer, this layer is not part of either the input or output. Note: Neural networks can have several hidden layers, in this example only one layer was included for simplicity. Finally, Layer 3 is the source layer. You may notice that there are many connections between Layer 1 (W1) and Layer 2 (W2). Each node in W1 has a connection with all nodes in W2, and from each node in W2 there is one connection to a single output node in W3. Each of these connections must have a corresponding weight.

All the mathematics above requires very precise notation. The notation used here is also used in the in-depth study guide from Stanford University. In the following equations, the weight of the corresponding bond will be denoted as wij (l), where i is the node number in layer l + 1, and j is the node number in layer l. For example, the weight of the bond between node 1 in layer 1 and node 2 in layer 2 will be denoted as w21 (1). I wonder why the indices 2-1 mean the relationship 1-2? This notation is clearer if you add an offset. The graph above shows that the +1 offset is associated with all nodes in the adjacent layer. The offset in W1 is related to all nodes in W2. Since the offset is not a real node with an activation function, it has no inputs (its input value is always equal to a constant). The weight of the connection between the offset and the node will be denoted by bi (l), where i is the node number in the layer l + 1, as well as in wij (l). For example from w21 (1), the weight between the offset in W1 and the second node in W2 will be denoted by b2 (1). Remember that these values ​​- wij (l) and bi (l) - will change during the learning process. The designation of the connection to the source node will look like this: hj (l), where j is the node number in layer l. Then in the previous example, the link to the source node is h1 (2)

As is clear from the previous section, the input data for the neural network should be reduced to the established form. What it means? Consider the following example: analyze the dynamics of the stock market. Prices in this case will be much higher than one. Therefore, you can reduce the data to the price difference, which will be expressed as a percentage. At the output we get a range of values ​​from -1 to 1. The described sequence of actions is called normalization of input data. This is the first and main step before starting machine learning. The system must receive the information in the form that it can process.

The next step is to obtain the first result of the calculations. In 99% of cases, it will be different from what it really should be. This fact is explained simply: the network does not have enough information for proper analysis (ie relevant weight distribution). At this stage, a learning algorithm is created - a training set. It is a set of operands that specify input processing parameters and help the neural network to estimate weight correctly. Depending on the complexity of the problem, from 4 to several hundred formulas can be used. The passage of the operand cycle will be called the epoch. At the time of creation, the neural network has an epoch numbered 0. After the first cycle of learning comes epoch 1, and so on. With each training cycle, the calculation error decreases. When this figure does not exceed a few percent, it is considered that the network has been trained and is suitable for solving real problems.
